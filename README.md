# Modern Article Card

## Overview

This project is a responsive article card built with HTML, CSS, and JavaScript. It demonstrates modern CSS techniques including CSS custom properties, OKLCH colors, fluid typography, responsive layout, and a light/dark mode toggle.

## Features

- Responsive article card
- Light and dark mode toggle
- Fluid typography using `clamp()`
- CSS custom properties (`:root`)
- OKLCH color space
- Accessible HTML
- Keyboard focus styles
- `text-box` property with fallback support

## Technologies Used

- HTML5
- CSS3
- JavaScript

## Running the Project

1. Clone or download the repository.
2. Open the project folder in Visual Studio Code.
3. Launch `index.html` using Live Server or Live Preview.
